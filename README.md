# 📝 Text Case Converter

A simple Python CLI tool that converts text into different letter cases.

## Features

- UPPERCASE
- lowercase
- Title Case
- Swap Case
- Capitalize

## Requirements

Python 3.x

## Run

```bash
python main.py
```

## Example

Input:

```
hello world
```

Output:

```
UPPERCASE:
HELLO WORLD

lowercase:
hello world

Title Case:
Hello World

Swap Case:
HELLO WORLD

Capitalize:
Hello world
```
